• header 태그 : 웹 문서나 섹션의 머리말을 정의하는 태그로, 로고, 제목, 내비게이션 메뉴 등을 포함합니다.
  • nav 태그 : 내비게이션 링크를 그룹화할 때 사용하는 태그로, 주요 메뉴나 링크 모음을 정의합니다.
  • section 태그 : 문서의 독립적인 콘텐츠 섹션을 나타내며, 주제별로 콘텐츠를 나눌 때 사용합니다.
  • aside 태그 : 문서의 주요 내용과 간접적으로 관련된 부가적인 콘텐츠(예: 광고, 사이드바)를 정의합니다.
  • footer 태그 : 문서나 섹션의 하단 부분을 정의하는 태그로, 저작권 정보, 연락처, 관련 링크 등을 포함합니다.
