# 문제2 : 출력하기 - 문자열연결 연산자

> str1과 str2를 연결하여 alert로 “hI ECMA Script”문자열을 출력하시오.

```javascript
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>2번</title>
  </head>
  <body>
    <script>
      let str1 = "Hi";
      let str2 = "ECMA Script";

      document.write(str1 + " " + str2);
```
